扫描日期: 2026-05-15 · 范围: GitHub 链上数据/回测/Agent/ML 四象限 · 目的: 把高价值开源能力嫁接到现有 v0.5 self-evolving agent

结论先行 · TL;DR
当前项目缺口集中在三处: ① 链上数据深度 (主要靠免费 API, 缺地址级粒度), ② 因子库广度 (10 因子 + Coinbase Premium 等 5 个, 行业基准是 Alpha158/Alpha360), ③ 自演化 Agent 的智能上限 (当前 narrative_tracker/factor_proposer 是单步 Claude 调用, 无多 agent 协作)。

推荐落地三件套: CryptoBERT NLP Alpha158 风格因子库 cryo 链上 adapter。这 3 项有共同特征——投入小 (~200-400 行)、风险低 (新增模块, 不破坏现有)、能立刻产生增量信号。
